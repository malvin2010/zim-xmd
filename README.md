## 🚀 Quick Deploy

### Deploy on Render (Recommended)
1. Fork this repo to your GitHub
2. Go to [render.com](https://render.com) → New → Web Service
3. Connect your GitHub repo
4. Set environment variables:
   - `OWNER_NUMBER` = Your WhatsApp number (e.g. `263XXXXXXXXX`)
5. Deploy — visit `your-app.onrender.com/pair` to pair

### Deploy on Panel / VPS
```bash
git clone https://github.com/malvinc/zim-xmd
cd zim-xmd
npm install
cp .env.example .env
# Edit .env with your details
node index.js
```

---

## 📱 Pairing the Bot

1. Open `http://localhost:3000/pair` (or your Render URL + `/pair`)
2. Choose **Pairing Code** tab
3. Enter your WhatsApp number with country code
4. Click **Get Pairing Code**
5. Open WhatsApp → Linked Devices → Link a Device → Link with phone number
6. Enter the 8-digit code
